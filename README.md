# MGTI - 英雄联盟五维游戏人格测试

MGTI，全称 **My Game Type Indicator**，是一个基于《英雄联盟》英雄设定与玩家游戏行为的趣味人格测试项目。

项目不再使用传统 16 型人格分类。当前版本采用 5 个连续维度描述玩家风格，并使用余弦相似度将用户画像与英雄画像进行匹配，最终生成一个更贴近游戏行为的本命英雄结果。

## ✨ 项目定位

MGTI 希望解决原有英雄人格测试中的三个问题：

- 类型过粗：固定人格类型难以区分同类英雄之间的差异。
- 游戏关联弱：传统人格字母和真实游戏行为之间的关系不够直接。
- 结果随机感强：单纯按类型抽取英雄，缺少可解释的匹配算法。

当前系统将玩家行为拆解为五个游戏维度。每个维度都使用 -2 到 +2 的连续分数表示倾向强弱。用户答题后得到一个五维向量，英雄也有预设的五维向量。系统通过向量相似度计算，选出与用户风格最接近的英雄。

## 🧠 MGTI 五维体系

| 维度代码 | 维度名称 | 低分倾向 | 高分倾向 | 游戏体现                               |
| -------- | -------- | -------- | -------- | -------------------------------------- |
| TAC      | 战术风格 | 直觉型   | 谋略型   | 对线时凭感觉还是计算技能冷却、换血收益 |
| TEA      | 团队角色 | 独行侠   | 团队核心 | 更喜欢单带还是支援队友                 |
| EMO      | 情绪反应 | 冷静     | 激情     | 逆风、被单杀、团战失误后的心态变化     |
| DEC      | 决策速度 | 谨慎     | 果敢     | 看到机会时等待时机还是立刻开打         |
| PRE      | 英雄偏好 | 传统     | 异类     | 偏好安全输出还是高风险高回报英雄       |

## 🎮 功能特性

- **在线交互测试**  
  用户回答题目后，系统自动生成五维游戏人格画像。

- **英雄向量匹配**  
  每个英雄都拥有 TAC、TEA、EMO、DEC、PRE 五维分数。系统通过余弦相似度计算匹配程度。

- **随机化前五名结果**  
  系统会先选出相似度最高的前 5 名英雄，再随机抽取 1 个作为最终结果。这样可以避免每次结果过于固定。

- **完整英雄图鉴**  
  图鉴页面展示所有英雄的基础资料、MGTI 标签和五维条形图。

- **可配置数据结构**  
  维度定义、测试题目、英雄五维分数、结果文案均通过 JSON 文件配置，便于后续调整。

- **纯前端部署**  
  项目可以直接部署到 GitHub Pages、Vercel、Netlify 或任意静态页面托管平台。
